# Sales Data Analysis Dashboard (Excel)

This project is an Excel-based Sales Dashboard created to analyze and visualize sales performance using interactive and easy-to-understand visuals.

The dashboard helps in understanding sales trends, performance by category, and overall business insights.

---

## Project Objective
The main goal of this project is to:
- Analyze sales data
- Create a clear and interactive dashboard
- Help users make better business decisions using data
